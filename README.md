standard-3min-check

― 3分で「気持ちを整理する」ための、評価しないチェック ―

これは何か？

standard-3min-check は、
「診断」や「評価」を行うツールではありません。

相談や行動に進む前段階として、
いまの状況や気持ちを整理するためだけの
シンプルなチェックツールです。

点数は出しません

タイプ分けもしません

正解・不正解はありません

ここで結論を出す必要はありません。

どんな場面で使うものか？

このツールは、以下のような場面を想定しています。

相談したい気持ちはあるが、何を話せばいいか分からない

決断を迫られているが、気持ちが追いついていない

「整理してから考えたい」と感じている状態

**行動を促すためのツールではなく、
“立ち止まることを許可するツール”**です。

想定利用シーン（例）

結婚相談所の公式サイト

学習塾・教育サービスの案内ページ

パーソナルトレーニング・コーチング

キャリア相談・カウンセリング

いずれも共通点は：

いきなり結論や契約に進ませない方が、信頼される場面

このツールが「しないこと」

多くのチェックツールがやりがちな、次のことは行いません。

❌ 診断結果のラベリング

❌ 優劣・適性・向き不向きの提示

❌ 行動・申込・契約の強制

❌ 不安を煽る表現

代わりに行うのは：

今の状態を短い言葉で整理する

気になっているポイントを順番付きで可視化する

次の一歩を選択肢として提示する

技術的な特徴（安心して使える理由）

Streamlit による軽量Webアプリ

生成AIは使用していません

個人情報の入力なし

ログイン・DB・外部保存なし

サーバー側でのデータ保持なし

つまり：

「診断された」「記録された」感覚が一切ありません。
